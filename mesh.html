class instaseis.mesh.Buffer(max_size_in_mb=100)[source]¶

A simple buffer class based on a dictionary with a maximum memory. Last accessed items are removed last when the memory limit is hit.

add(key, value)[source]¶

Add an item to the buffer and make sure that the buffer does not exceed the maximum size in memory.

efficiency()[source]¶

Return the fraction of call to the __contains__() routine that returned True.

get(key)[source]¶

Return an item from the buffer and move it to the end, so it is removed last.
