InstaSeisDB¶

class instaseis.instaseisdb.InstaSeisDB(db_path, buffer_size_in_mb=100, read_on_demand=True, reciprocal=True)[source]¶

A class to extract Seismograms from a set of wavefields generated by AxiSEM. Taking advantage of reciprocity of the Green’s function, two simulations with single force sources (vertical and horizontal) to build a complete Database of Green’s function in global 1D models. The spatial discretization equals the SEM basis functions of AxiSEM, resulting in high order spatial accuracy and short access times.

Parameters:
  • db_path (str) – Path to the AxiSEM Database containing subdirectories PZ and/or PX each containing a order_output.nc4 file
  • buffer_size_in_mb (int, optional) – Strain is buffered to avoid unnecessary file access when sources are located in the same SEM element
  • read_on_demand (bool, optional) – read several global fields on demand (faster initialization, default) or on initialization (slower initialization, faster in individual seismogram extraction, useful e.g. for finite sources)
  • reciprocal (bool, optional) – assume a reciprocal database (fixed receiver depth, sources anywhere) or a forward database (fixed source depth, receiver anywhere)
get_seismograms(source, receiver, components=('Z', 'N', 'E'), remove_source_shift=True, reconvolve_stf=False, return_obspy_stream=True, dt=None, a_lanczos=5)[source]¶

Extract seismograms for a moment tensor point source from the AxiSEM database.

Parameters:
  • source (instaseis.source.Source or instaseis.source.ForceSource) – instaseis.Source or instaseis.ForceSource object
  • receiver (instaseis.source.Receiver) – instaseis.Receiver object
  • components – a tuple containing any combination of the strings "Z", "N", "E", "R", and "T"
  • remove_source_shift – move the starttime to the peak of the sliprate from the source time function used to generate the database
  • reconvolve_stf – deconvolve the source time function used in the AxiSEM run and convolve with the stf attached to the source. For this to be stable, the new stf needs to bandlimited.
  • return_obspy_stream – return format is either an obspy.Stream object or a plain array containing the data
  • dt – desired sampling of the seismograms. resampling is done using a lanczos kernel
  • a_lanczos – width of the kernel used in resampling

Extract seismograms for a finite source from the AxiSEM database provided as a list of point sources attached with source time functions and time shifts.

Parameters:
  • sources (list of instaseis.source.Source objects) – A collection of point sources.
  • receiver (instaseis.source.Receiver) – The receiver location.
  • components – a tuple containing any combination of the strings "Z", "N", and "E"
  • dt – desired sampling of the seismograms.resampling is done using a lanczos kernel
  • a_lanczos – width of the kernel used in resampling
  • correct_mu – correct the source magnitude for the actual shear modulus from the model
